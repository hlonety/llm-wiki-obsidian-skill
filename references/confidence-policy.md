# Confidence Policy

Confidence is not a popularity counter. It is a guarded quality signal that helps prevent weak or stale claims from compounding into accepted knowledge.

## Source Count Rules

Use `source_count` for external, independent sources only.

Do not count:

- Personal writing under `raw/personal/`.
- The user's own essays, investment notes, draft opinions, or analysis reports.
- Query outputs generated by the agent.
- Synthesis pages created from already-counted sources.

Personal writing can establish the user's position, but it should not make a concept look better supported by external evidence.

## Confidence Ladder

| Evidence | Confidence | Rule |
| --- | --- | --- |
| 0 sources | low | Use only for seed notes, questions, or personal hypotheses. |
| 1 external source | low | Single-source claims remain fragile. |
| 3+ external sources | medium | Allowed when sources broadly agree and provenance is clear. |
| 5+ external sources | candidate-high | Ask the user to review definition, sources, and contradictions. |
| User confirms | high | Only after explicit human confirmation. |

Never promote to `confidence: high` automatically.

## Candidate High Review

When a page reaches 5+ independent external sources and has no major contradiction, ask:

```text
This page has 5+ external sources and no unresolved contradiction. Please review the definition and source list. Should confidence be promoted to high?
```

Only set:

```yaml
confidence: high
high_confirmed: true
high_confirmed_by: user
high_confirmed_on: YYYY-MM-DD
```

after explicit confirmation.

## Volatility and Staleness

Use `domain_volatility` for pages whose facts age differently:

```yaml
domain_volatility: high    # review after 90 days
domain_volatility: medium  # review after 180 days
domain_volatility: low     # review after 365 days
last_reviewed: YYYY-MM-DD
```

Examples:

- `high`: model benchmarks, API pricing, tool capabilities, regulations.
- `medium`: current best practices, market structure, active research debates.
- `low`: historical papers, stable definitions, biographies.

Stale does not mean false. It means the page should not be treated as settled until reviewed.

## Contradictions

When sources disagree:

```yaml
status: contested
contested: true
confidence: low
contradictions: [other-page-or-source]
```

Do not hide disagreement by averaging claims. Record the disagreement, dates, and sources.
