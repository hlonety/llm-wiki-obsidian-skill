# LLM Wiki

An LLM Wiki is a Markdown knowledge base where raw sources are preserved separately from compiled synthesis pages. It differs from ordinary retrieval because durable pages such as [[llm-wiki]] and [[ai-learning-map]] are improved over time instead of being recreated from scratch for every question.

The useful memory is not just the text stored on disk. The useful memory is the set of editorial decisions encoded into pages: which concepts deserve their own page, which sources support each claim, what contradictions remain unresolved, and which questions should guide the next ingest. This makes the vault a working research surface instead of a static clipping folder.

For AI learning, the pattern is especially valuable because tools, models, benchmarks, and best practices change quickly. Pages should track confidence, source count, domain volatility, and review dates so the user can tell the difference between stable foundations and fast-moving claims that need another check.
